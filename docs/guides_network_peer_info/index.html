Network Peer Info

Overview

Get Network Peer command shows detailed info about nodes with passed public addresses. Let's imagine that you want to get detailed info about nodes with public addresses like 14F8423888AE5F719B06FF5B5B242DDD31CE8003639FC69A2D3EFB82E32A2FBF and 73472A2E9DCEA5C2A36EB7F6A34A634010391EC89E883D67360DB16F28B9443C.

Shell
Go
dfms-cli network peer 14F8423888AE5F719B06FF5B5B242DDD31CE8003639FC69A2D3EFB82E32A2FBF 73472A2E9DCEA5C2A36EB7F6A34A634010391EC89E883D67360DB16F28B9443C
