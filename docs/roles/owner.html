Owner

Overview

Owner is a user that wants storing files in DFMS. It create drive account and prepare it.

What do

  1. Creating drive account that will using replicators.
  2. Creating a drive with the desired properties.
  3. Interacting with drive (uploading, removing, renaming, moving, downloading files and directories)
  4. Initiating challenge for checking if replicators store files
